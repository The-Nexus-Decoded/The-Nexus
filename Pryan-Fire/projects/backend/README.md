# Backend Projects — Haplo's Domain

Backend service projects: APIs, process supervisors, CI/CD tooling, agent infrastructure, webhook handlers.

## What Goes Here

- Backend service architecture specs
- API design documents
- Infrastructure automation plans
- CI/CD pipeline designs
- Agent tooling specs

## What Does NOT Go Here

- Trading algorithms (goes in `trading/`)
- Frontend/UI (goes in `Arianus-Sky/projects/design/`)
- Fleet orchestration (goes in `Chelestra-Sea/projects/fleet/`)
