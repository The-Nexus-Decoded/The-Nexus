# QA Projects — Marit's Domain

Quality assurance test plans, audit frameworks, performance benchmarks, acceptance criteria, quality gates.

## What Goes Here

- Test plan specifications per project
- Acceptance criteria documents
- Performance benchmark frameworks
- Security audit checklists
- Cross-realm quality gate definitions
- Bug triage and regression tracking

## What Does NOT Go Here

- Test code/scripts (goes in the realm being tested, e.g., `Pryan-Fire/tests/`)
- Security infrastructure (goes in `Chelestra-Sea/projects/fleet/`)
- Project specs (goes in the appropriate realm's project folder)
