# Mobile Projects — Paithan's Domain

Mobile application projects: iOS, Android, cross-platform (React Native, Flutter), mobile WebXR fallbacks.

## What Goes Here

- Mobile app specs and architecture plans
- Platform-specific requirements (HIG, Material Design)
- API surface definitions (what mobile consumes from backend)
- Feature priority lists and MVP scoping
- Companion app specs for XR projects

## What Does NOT Go Here

- Web-only UI projects (goes in `design/`)
- VR/XR core experiences (goes in `games-xr/`)
- Backend APIs (goes in `Pryan-Fire/`)
- App store marketing (goes in `Chelestra-Sea/projects/growth/`)
