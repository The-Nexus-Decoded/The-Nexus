# Design Projects — Orla's Domain

UI/UX design projects: design systems, component libraries, visual language, accessibility audits, and design specs handed to engineering.

## What Goes Here

- Design system specifications
- Component library specs (tokens, patterns, variants)
- Accessibility audit reports
- Visual language guides per project
- Mockup and wireframe references

## What Does NOT Go Here

- Source code (goes in `Arianus-Sky/src/`)
- Mobile-specific projects (goes in `mobile/`)
- VR/XR/game design (goes in `games-xr/`)
- Backend logic (goes in `Pryan-Fire/`)
