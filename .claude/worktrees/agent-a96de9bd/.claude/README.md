# Claude Code Session Directory

This directory stores session handoffs and continuity files for Claude Code when operating on The-Nexus monorepo.

- `handoffs/` — Markdown files capturing session state for resumption
- `prompt.txt` — Optional base context (merged with repo's GEMINI.md)

This folder is created to satisfy auto-initialization requirements. Actual handoff files are generated automatically by the agent during session transitions.

Do not edit manually unless intentionally crafting a handoff.
